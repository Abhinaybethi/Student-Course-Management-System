# Student Course Management System (SQL Project)

A simple SQL project that manages students, courses, and enrollments.  
Includes relational database design, sample data, and analytics queries.

## Database Features
- Students information
- Courses and credits
- Enrollment management
- Grade tracking
- Analytical SQL queries

## Technologies Used
- MySQL / MariaDB / PostgreSQL

## Files
- schema.sql → Creates all tables
- insert_data.sql → Inserts sample records
- queries.sql → 15+ useful SQL queries

## Example Queries
- Students in CSE
- Courses taken by a student
- Count enrollments per course
- Top students with grade A
- Students with no course enrolled
